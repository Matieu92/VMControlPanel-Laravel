# VM-CONTROL

Profesjonalny system zarządzania instancjami wirtualnymi i infrastrukturą serwerową, <br> realizowany w laravelu jako zaliczenie przedmiotu Aplikacje Internetowe.

## Kluczowe cechy projektu

* **Unikatowy Interfejs**: Autorski wygląd strony zaprojektowany w stylu produkcyjnym.
* **Pełna Dostępność (WCAG 2.1)**:
    * Tryb wysokiego kontrastu (Contrast Mode).
    * System powiększania czcionek bez rozbijania układu.
    * Pełna obsługa czytników ekranu poprzez atrybuty ALT i role ARIA.
* **Architektura Bazy Danych**: Obsługa różnych systemów baz danych.
* **System Finansowy**: Moduł portfela z historią transakcji i walidacją doładowań.
* **Zarządzanie Zasobami**: Konfigurator serwerów, zarządzanie węzłami (Nodes) oraz systemami operacyjnymi.

## Technologie

* **Backend**: Laravel 11 / PHP 8.x
* **Frontend**: Blade / Custom CSS

## Instalacja

1. Sklonuj repozytorium.
2. Wykonaj `composer install`.
3. Skonfiguruj `.env`.
4. Uruchom migracje: `php artisan migrate --seed`.
5. Wygeneruj klucz: `php artisan key:generate`.
